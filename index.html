Instinct

Instinct is a service that translates text messages in real-time, that uses Flask.py as a python framework and SQL as a database. allowing for a seamless conversation in each user’s native language. Preferred languages are automatically recognized based on each user’s outbound messages, and all incoming messages are automatically translated and displayed in that language.
